# Static files (web)

Files in this folder are served by the Expo dev server when running `npx expo start --web`.

- **favicon.ico** — If you add a `favicon.ico` file here, the browser tab will show it and the `:8081/favicon.ico` console error can be avoided. You can copy your app icon (e.g. from `assets/brand/`) and rename or convert it to `favicon.ico`.
