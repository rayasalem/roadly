# لوحة تحكم المزود — Wireframe / Mockup كامل (جاهز لـ Figma و React Native)

لوحة تحكم موحّدة لكل مزود خدمة سيارات (ميكانيكي، ونش، مؤجر سيارات): رأس الصفحة، شريط جانبي، الطلبات، التوافر، الإحصائيات، الملف، المساعدة.

---

## 1. الهيكل العام

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  [صورة/شعار]   اسم المزود              [🔔 2]  [تسجيل خروج]  ← Header (خلفية #000)  │
├─────────────────────────────────────────────────────────────────────────────────┤
│  ┌──────────────┐ │                                                              │
│  │ 🏠 الرئيسية  │ │   [محتوى حسب القسم المختار]                                  │
│  │ 📋 الطلبات   │ │   — لوحة الطلبات: توافر + طلبات جديدة + جارية + مكتملة       │
│  │ 📜 السجل     │ │   — أو الطلبات الواردة فقط                                    │
│  │ 📊 الإحصائيات│ │   — أو السجل والطلبات المنفذة                                 │
│  │ ⚙️ الملف     │ │   — أو الإحصائيات والرسم البياني                             │
│  │ ❓ المساعدة  │ │   — أو الملف والإعدادات                                        │
│  └──────────────┘ │   — أو المساعدة والدعم الفني                                  │
│  Sidebar (عرض≥600)   Main content (خلفية #F6F6F6)                               │
└─────────────────────────────────────────────────────────────────────────────────┘
```

على الشاشات الضيقة (موبايل): بدل الشريط الجانبي — **شريط تبويبات أفقي** (نفس الستة أقسام) فوق المحتوى.

---

## 2. رأس الصفحة (Header) — مواصفات دقيقة

| العنصر | المواصفات |
|--------|------------|
| الخلفية | `#000000` |
| الارتفاع | 56px كحد أدنى |
| من اليسار | صورة/شعار المزود: دائرة 44×44، خلفية `rgba(255,255,255,0.2)`، أيقونة account-wrench 32px أبيض (أو صورة المستخدم إن وُجدت) |
| بجانب الصورة | اسم المزود: 18px Semibold، لون `#FFFFFF`، سطر واحد مع اختصار |
| من اليمين | أيقونة إشعارات: bell-outline 24px أبيض، مع **عداد** (Badge) عند وجود طلبات جديدة: دائرة حمراء `#EF4444`، نص العدد أبيض 11px |
| أقصى اليمين | زر تسجيل خروج: أيقونة logout 24px أبيض |

---

## 3. شريط التنقل (Sidebar / Nav)

- **عرض الشريط الجانبي**: 200px، خلفية `#FFFFFF`، حد يمين `#E5E7EB`.
- **كل عنصر**: صف أفقي — أيقونة 22px، نص 14px. المسافة العمودية 12px، الأفقي 16px.
- **العنصر النشط**: خلفية `rgba(34,197,94,0.1)`، لون النص والأيقونة `#22C55E`.
- **العنصر غير النشط**: لون النص والأيقونة `#6B7280`.

قائمة العناصر:

| الأيقونة | المفتاح (i18n) | التسمية (عربي) |
|----------|----------------|-----------------|
| view-dashboard-outline | sidebar.dashboard | الرئيسية |
| clipboard-list-outline | sidebar.requests | الطلبات الواردة |
| history | sidebar.history | السجل والطلبات المنفذة |
| chart-bar | sidebar.statistics | الإحصائيات |
| account-cog-outline | sidebar.profile | الملف والإعدادات |
| help-circle-outline | sidebar.help | المساعدة |

**على الموبايل**: شريط أفقي قابل للتمرير، كل تبويب: خلفية `#F6F6F6`، حدود، النشط: خلفية `#22C55E` ونص أبيض.

## 4. لوحة الطلبات (قسم الرئيسية — Dashboard)

### 4.1 حالة التوافر (أعلى المحتوى)

- **بطاقة واحدة**: خلفية خضراء `#22C55E` عند "متاح"، حمراء `#EF4444` عند "غير متاح".
- **المحتوى**: نقطة بيضاء 12×12، نص "متاح للطلبات" أو "غير متاح" (18px Bold أبيض)، سطر فرعي (12px أبيض شفاف)، وزر لتبديل الحالة: "غير متاح" أو "متاح" (خلفية شبه شفافة بيضاء، نص أبيض).
- **الارتفاع**: padding 20 عمودي، borderRadius 16، ظل خفيف.

### 4.2 الطلبات الجديدة

- **عنوان القسم**: "طلبات جديدة" — 16px Semibold، لون `#111827`.
- **بطاقة طلب واحدة** تحتوي على:
  - **خريطة مصغرة**: ارتفاع 72px، خلفية `#E5E7EB`، أيقونة map-marker 28px لون `#22C55E`، وعنوان الموقع (12px، لون ثانوي) في سطر واحد.
  - **جسم الطلب**: اسم العميل (16px Semibold)، سطر تحته: نوع الخدمة · وقت الطلب (14px، لون ثانوي).
  - **أزرار الإجراء**: زر **قبول** (خلفية `#22C55E`، أيقونة check، نص "موافقة" أبيض)، زر **رفض** (خلفية `#EF4444`، أيقونة close، نص "رفض" أبيض). نفس العرض (flex 1)، padding 10 عمودي، borderRadius 12.

### 4.3 الطلبات جاري التنفيذ

- **عنوان**: "جاري التنفيذ".
- **بطاقة**: اسم العميل، نوع الخدمة، حالة "جاري التنفيذ" أو "قريب الانتهاء"، وروابط/زر "عرض على الخريطة" (أيقونة + نص بلون أخضر).

### 4.4 الطلبات المكتملة

- **عنوان**: "مكتملة".
- **بطاقة**: اسم العميل، نوع الخدمة، التاريخ، وتقييم العميل (نجوم أو رقم) إن وُجد — أيقونة نجمة 16px لون `#F59E0B`.

---

## 5. قسم الطلبات الواردة (Incoming Requests)

- نفس تصميم "الطلبات الجديدة" أعلاه: قائمة بطاقات طلب، كل بطاقة تحتوي خريطة مصغرة، تفاصيل العميل ونوع الخدمة والوقت، وأزرار قبول (أخضر) ورفض (أحمر).

---

## 6. قسم السجل (History)

- **عنوان**: "السجل والطلبات المنفذة".
- **قائمة بطاقات**: لكل طلب منفذ — اسم العميل، نوع الخدمة، التاريخ، وتقييم العميل (نجوم/رقم) إن وُجد.

---

## 7. قسم الإحصائيات (Statistics)

- **عنوان**: "الإحصائيات".
- **شبكة أرقام**: ثلاثة صناديق (اليوم، هذا الأسبوع، هذا الشهر) — رقم كبير (18px Bold)، وتسمية أسفل (12px لون ثانوي). حدود، borderRadius 12.
- **صفوف إضافية**: "إجمالي الأرباح" (قيمة بلون أخضر)، "معدل القبول" (نسبة مئوية)، "معدل الرفض" (إن وُجد).
- **منطقة الرسم البياني**: ارتفاع 180px، خلفية `#F6F6F6`، حدود، أيقونة chart-line 48px في المنتصف، ونص توضيحي "رسم بياني — الطلبات / الأرباح" (يمكن استبدالها برسم Bar أو Line فعلي لاحقاً).

---

## 8. قسم الملف والإعدادات (Profile & Settings)

- **عنوان**: "الملف والإعدادات".
- **أزرار**: "الملف الشخصي" (ينتقل لشاشة Profile)، "الإعدادات" (ينتقل لشاشة Settings). أزرار outline، عرض كامل.

---

## 9. قسم المساعدة (Help)

- **أيقونة**: help-circle-outline 48px لون ثانوي.
- **عنوان**: "المساعدة والدعم الفني".
- **وصف**: نص توضيحي (مثلاً: "تواصل مع الدعم للحصول على المساعدة الفنية.").

## 11. قائمة مكونات Figma (نسخ مباشر)

1. **Header** — شريط أسود، صورة 44×44، اسم، أيقونة إشعارات + Badge، أيقونة خروج.
2. **Sidebar** — عمود 200px، 6 عناصر (أيقونة + نص)، تمييز العنصر النشط.
3. **NavTabs (موبايل)** — شريط أفقي، 6 تبويبات، التبويب النشط أخضر.
4. **AvailabilityCard** — بطاقة خضراء/حمراء، نقطة، نص، زر تبديل.
5. **RequestCard** — خريطة مصغرة 72px، اسم عميل، خدمة · وقت، زر قبول، زر رفض.
6. **InProgressCard** — اسم، خدمة، حالة، رابط خريطة.
7. **CompletedCard** — اسم، خدمة، تاريخ، تقييم نجوم.
8. **StatsGrid** — 3 صناديق (اليوم، أسبوع، شهر).
9. **StatRow** — تسمية + قيمة (أرباح، معدل قبول).
10. **ChartPlaceholder** — منطقة 180px مع أيقونة ونص.
11. **ProfileSection** — زران: الملف، الإعدادات.
12. **HelpSection** — أيقونة، عنوان، وصف.
