# وثائق التصميم والمواصفات — تطبيق مزودي خدمات السيارات

## النسخة المحسّنة والمتقدمة (مرجع واحد)

**[APP-FULL-IMPROVED-SPEC.md](./wireframes/APP-FULL-IMPROVED-SPEC.md)** — مواصفات و Mockup/Wireframe **كامل** يجمع كل التحسينات التالية، جاهز للنسخ على Figma أو React Native:

1. **لوحة التحكم للمزود** — أقسام الطلبات (جديدة، جارية، مكتملة)، التوافر، الإشعارات، الخريطة، الإحصائيات والتقييمات.
2. **تحسين الخريطة** — أيقونات حسب المهنة (ميكانيكي/ونش/تأجير)، Popup، تحديث المواقع، عدم تراكب الماركرات.
3. **إدارة حالة الطلبات** — كل الحالات بألوان موحّدة (جديد، قيد الانتظار، مقبول، مرفوض، جاري التنفيذ، مكتمل، ملغى).
4. **نظام التقييم** — التقييم العام، السرعة، الجودة، الاحترافية، تعليق؛ عرض للمزود والعميل ومتوسط على الخريطة.
5. **تحسين UX** — فلترة، مفضلين، طلب مسبق (الوقت المفضل)، تتبع على الخريطة، إلغاء/تعديل الطلب.
6. **الإشعارات الذكية** — للمزود والعميل مع تخصيص تشغيل/إيقاف حسب النوع.
7. **خيارات إضافية** — دردشة، رفع صور/فيديو، خيارات دفع، عروض وخصومات.
8. **التصميم البصري** — خلفية فاتحة، ألوان متناسقة، نصوص كبيرة، أيقونات وترتيب منطقي للأقسام.
